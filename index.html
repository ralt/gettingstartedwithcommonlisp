Getting started with Common Lisp

Introduction

This webpage is a comprehensive list of resources for people new in the Common Lisp (CL) world. It tries its best to :

  • Define the terms used in the CL world.
  • Show the best resources to learn about CL.
  • Learn what the common tools in the CL world are.

Glossary

Common Lisp:
Common Lisp, usually abbreviated as CL (and not clisp), is a language defined by an ANSI standard.
REPL:
read-eval-print-loop, an interpreter. Quite useful for interactive development.
Implementation:
An standard-conforming implementation of Common Lisp should be able to compile/interpret standard-conforming Common Lisp code.
SBCL and Clozure CL are some free implementations.

Resources

This is a list of community-recognized resources you will need to make your way around CL.

Practical Common Lisp
Learn Common Lisp from the ground up. This is the resource that everyone recommends to people wanting to get started with Common Lisp.
On Lisp
On Lisp is probably the book with the best coverage on CL macros. And Paul Graham is an interesting man.
HyperSpec
This is probably the most complete readable documentation about Common Lisp.

Tools

QuickLisp
Quicklisp has, in recent times, become the de-facto method of obtaining open source Lisp libraries.
ASDF
Another System Definition Facility (ASDF) is a commonly used build system in the Lisp world.
Slime
Superior Lisp Interaction Mode for Emacs (SLIME) is a commonly used Emacs mode that turns Emacs into a Lisp IDE.
Slimv
Think SLIME, but for Vim users.
