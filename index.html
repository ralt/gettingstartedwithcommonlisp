Getting started with Common Lisp

List of resources to help you get started with Common Lisp

Introduction

This webpage is a comprehensive list of resources for people new in the Common Lisp (CL) world. It tries its best to :

  • Define some terms used in the CL world.
  • Show the best resources to learn about CL.
  • Learn what the common tools in the CL world are.

Glossary

Here are some common terms used in the CL world. This is in no way an exhaustive list, it is just here to give you an introduction.

Common Lisp
Common Lisp (abbreviated CL) is a language defined by a standard.
REPL
Read-Eval-Print-Loop. A tool that lets you write code and run it right away.
CL Implementation
A CL implementation is a software that can read and compile CL code according to the CL standard. You will need one of these if you want to run your code.
Some examples are SBCL, CLISP, ABCL, Clozure CL, CMUCL, ECL or LispWorks.

Resources

This is a list of community-recognized resources you will need to make your way around CL.

Practical Common Lisp
Learn Common Lisp from the ground up. This is the resource that everyone recommends to people wanting to get started with Common Lisp.
On Lisp
On Lisp is probably the book with the best coverage on CL macros. And Paul Graham is an interesting man.
HyperSpec
This is probably the most complete readable documentation about Common Lisp.

Tools

There are some tools that everybody writing code with CL uses. Here are some of them.

QuickLisp
QuickLisp is the community-recognized package manager for CL libraries. Look at its introduction to see how easy it is to install packages.
ASDF
ASDF is the make of Common Lisp. It allows you to specify how your Lisp source code is organized and should be compiled.
Slime
Slime is a tool you use with Emacs, the text editor. It gives you a complete IDE within Emacs, with an REPL, a debugger, auto completion, and much more.
Slimv
Slimv is the Slime for Vim users.
